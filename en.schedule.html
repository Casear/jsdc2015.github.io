PgREST - PostgreSQL, JavaScript, and RESTPgREST turns PostgreSQL into a RESTful JSON document store, allows running JavaScript and npm modules as stored procedures for queries and triggers from within the database. It also provides Firebase-compatible API for developing real-time applications. This talk will cover the building blocks of PgREST: PostgreSQL, plv8js, plv8x, as well as examples for turning existing relational data into REST endpoints.

Greg Weng

Functional JavaScript, why or why not?While some people feel functional programming is just an impractical buzz word, developers are taking actions to make certain nice or even weird features available on those not-so-functional languages. Could we play these new tricks well with JavaScript, the most misunderstood language in the world? Or would we get into trouble, while we're trying to escape from the deepest monad? In this talk, we would begin our journey from the simplest high-order functions, to see if these functional features can be successfully applied in JavaScript programming.

Feature-oriented Programming with JavaScript特色導向程式開發（找不到中文翻譯，所以暫譯） - Feature-oriented Programming (FOP) 是一個不同於傳統物件導向(OOP)的程式開發概念，在近年來慢慢開始有人討論其可行性。可以有效率擴充和增加程式功能，相當具有彈性，並易於管理程式架構和邏輯，利於軟體產品的開發。
某一天，不小心引入 FOP 到 JavaScript 和 Node.js 的開發後，意外發現 FOP 與動態語言相當契合，別有一番風味，可以用完全不一樣的方法，來看待和開發 JavaScript 應用程式。FOP 奇幻冒險，就此展開。
